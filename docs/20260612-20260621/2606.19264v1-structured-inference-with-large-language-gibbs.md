## 摘要
大型语言模型（LLM）中编码的知识可以作为描述复杂世界变量的结构化推理基础，但以概率一致的方式访问这些知识构成了一个困难的推理问题。我们提出了大型语言吉布斯（Large Language Gibbs）方法，这是一种结构化概率推断方案，它利用LLM的条件分布作为转移算子。不同于通过单次自回归生成来采样结构化对象，我们使用LLM的下一个标记条件分布，在给定其他变量的条件下迭代地重新采样单个变量。这种方法避免了顺序依赖偏差，并产生一个反映所有局部条件分布之间折中的平稳分布。我们将该方法应用于从合成分布中采样、一致性推理任务和贝叶斯结构学习。结果表明，在通过含噪声的LLM条件分布可访问的世界先验下，使用MCMC中的LLM条件分布是用于结构化概率推断的单次生成的一种实用替代方案。

## Abstract
The knowledge encoded in large language models (LLMs) can serve as a substrate for structured reasoning over variables describing a complex world, but accessing this knowledge in a probabilistically coherent manner poses a difficult inference problem. We propose Large Language Gibbs, a scheme for structured probabilistic inference that uses conditional distributions of an LLM as transition operators. Rather than sampling structured objects through single-pass autoregressive generation, we iteratively resample individual variables conditioned on others using an LLM's next-token conditionals. This approach avoids order-dependent biases and produces a stationary distribution that reflects a compromise between all local conditionals. We apply this approach to sampling from synthetic distributions, consistent reasoning tasks, and Bayesian structure learning. The results suggest that the use of LLM conditionals in MCMC is a practical alternative to one-pass generation for structured probabilistic inference under a world prior accessible through noisy LLM conditionals.

## 1. 核心问题与整体含义（研究动机与背景）
- **问题**：大型语言模型（LLM）被越来越多地用于表示结构化对象（如标注分配、表格数据、组合对象）的分布。然而，传统的单遍自回归生成方法存在**顺序依赖偏差**（如近因效应、忽略中间上下文），导致生成的联合分布扭曲。如何以概率一致的方式从LLM中提取结构化的联合知识是一个难题。
- **背景**：现有方法（如顺序解码、批采样）要么在边际分布上不准确，要么样本间存在高自相关。同时，受掩码语言模型（如BERT）作为Gibbs转移算子的启发，作者试图将自回归LLM的条件分布用于马尔可夫链蒙特卡洛（MCMC）中的迭代重采样，以克服固定生成顺序带来的偏差。
- **整体含义**：提出**Large Language Gibbs**框架，将LLM的条件分布作为Gibbs核，迭代重采样变量，其平稳分布反映了所有局部条件分布的折中，为结构化概率推断提供了一种比单次生成更概率一致且实用的替代方案。

## 2. 方法论：核心思想、技术细节与算法流程
### 2.1 核心思想
- 将一组变量 \(X = (X_1, \dots, X_n)\) 视为结构化对象，每个变量有自然语言描述 \(c_i\)。利用LLM的next-token条件分布来近似每个变量的全条件分布 \(q_i(X_i | X_{-i})\)，然后通过Gibbs采样迭代重采样，最终达到一个平稳分布 \(q^*\)（即使局部条件分布不兼容）。
- 关键：重采样时随机排列其他变量的顺序，避免固定顺序偏差。

### 2.2 关键技术细节
- **基本版本（§3.1）**：
  - 定义重采样核：  
    \(q^{\text{LM}}_i(X_i | X_{-i}) = \mathbb{E}_{\sigma_{-i}} [p^{\text{LM}}(X_i | \text{seq}(X_{-i}, \sigma_{-i}) \oplus [c_i])]\)，其中 \(\sigma_{-i}\) 是其他变量序列的随机排列。
  - 算法流程：初始化变量（如自回归生成）→ 每步均匀选择变量 \(i\) → 随机排列其余变量 → 用LLM生成 \(X_i\) → 重复。使用烧入期（burn-in）和稀疏化（thinning）。
  - **平稳性分析**：在假设“联合似然在最后变量位置上对称”（Assumption 3.1）下，该核的平稳分布等于所有排列平均的联合分布 \(q^{\text{sym}}\)（Proposition 3.2）。该假设弱于完全排列不变性。

- **判别式预测核（§3.2）**：对于指令微调模型，生成式条件可能校准不良。提出两种Metropolis-within-Gibbs变体：
  - **Barker Gibbs**：从均匀提议中抽取候选，用LLM判断候选与当前值谁更合理，采用Barker acceptance概率。
  - **Gambling Gibbs**：让LLM决定是否投注（基于随机金额），接受投注则接受候选。只需贪婪解码，不依赖校准概率。

- **实践变体（§3.3）**：
  - **Sweeping Gibbs**：固定顺序循环重采样（而非随机选取），可能加速收敛但平稳分布可能变化。
  - **Block Gibbs**：同时重采样一组变量，帮助逃离局部模式。

## 6. 主要结论与发现
1. **简单分布采样**：独立采样偏差大、批采样自相关高；Gibbs采样（尤其是基本版和Block Gibbs）显著同时改善了偏差和自相关。指令微调会引入额外偏差，但Gambling Gibbs配合大模型+思维链可部分缓解。
2. **一致性推理**：提出的Gibbs方法（n=64）明显优于零样本、自回归和ICM基线；随着n增大（从4到64），性能单调提升，证明迭代重采样和随机排序的有效性。
3. **贝叶斯结构学习**：利用LLM合成数据作为先验（通过Gibbs生成）多数情况下优于均匀先验；Direct方法有时失败（如小数据集），而Gibbs方法更稳定。重要条件是元数据信息丰富，否则可能恶化（如algalactivity2失败案例）。
4. **理论保证**：在弱对称性假设下，Gibbs核的平稳分布等于排列平均联合分布，提供解释。
5. **实用建议**：基本Gibbs当LLM生成校准好时适用；指令模型配合Barker/Gambling Gibbs可绕过校准问题。
