## 摘要
世界模型提供了一种原则性的方法，使长程LLM智能体具备预见能力：在执行前预测行动后果。然而，不可靠的预见可能被忽略、误用，甚至损害下游决策。在本文中，我们引入了WorldEvolver，一个自我进化的世界模型框架，它在保持下游智能体和所有模型参数冻结的同时，修正其部署时上下文。WorldEvolver集成了三个模块：（i）情景记忆，通过基于检索的模拟利用真实的行动转换；（ii）语义记忆，从预测-观察不匹配中提取持久性启发规则；（iii）选择性预见，在将低置信度预测整合到智能体推理上下文之前对其进行过滤。我们在ALFWorld和ScienceWorld上评估WorldEvolver，在Word2World上测量世界模型预测准确率，在AgentBoard上测量下游智能体成功率。大量实验表明，WorldEvolver在三个骨干网络上取得了最高的预测准确率，并在下游智能体成功率上领先其他世界模型基线，表明测试时记忆修正提高了预测保真度和规划性能。

## Abstract
World models offer a principled way to equip long-horizon LLM agents with foresight: predictions of action consequences before execution. However, unreliable foresight can be ignored, misused, or even degrade downstream decision-making. In this paper, we introduce WorldEvolver, a self-evolving world model framework that revises its deployment-time context while keeping the downstream agent and all model parameters frozen. WorldEvolver integrates three modules: (i) Episodic Memory, which exploits real action transitions through retrieval-based simulation; (ii) Semantic Memory, which extracts persistent heuristic rules from prediction-observation mismatches; and (iii) Selective Foresight, which filters low-confidence predictions before integrating them into agent reasoning context. We evaluate WorldEvolver on ALFWorld and ScienceWorld, measuring world model prediction accuracy on Word2World and downstream agent success rate on AgentBoard. Extensive experiments show that WorldEvolver achieves the highest prediction accuracy across three backbones and leads other world model baselines on downstream agent success rate, demonstrating that test-time memory revision enhances both predictive fidelity and planning performance.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：世界模型为LLM智能体提供“预见”（foresight），即预测行动后果，从而改善长程规划。然而，部署环境会不断变化（分布偏移），导致冻结的世界模型预测不可靠；而不可靠的预见可能被智能体忽视、误用甚至损害下游决策。
- **研究动机**：现有方法要么冻结世界模型（无法适应新分布），要么通过参数更新（高计算成本、可能造成灾难性遗忘）。因此需要一种**自我进化**的世界模型，在部署期间根据实际交互反馈不断修正，而不更新模型权重。
- **整体意义**：本文提出WorldEvolver，一种**训练自由**的框架，通过测试时记忆修订（test-time memory revision）同时提升世界模型预测保真度和智能体规划成功率。证明了**可靠预见的关键不在于参数更新，而在于如何动态组织并提供上下文证据**。

## 2. 论文提出的方法论：核心思想、关键技术细节、算法流程

### 核心思想
- **冻结世界模型和智能体参数**，只通过**非参数记忆**（非参数记忆存储）在外循环中进化，避免在线更新带来高成本和副作用。
- 记忆分为两类：**情景记忆**（Episodic Memory）存储具体交互经验，用于检索相似转换；**语义记忆**（Semantic Memory）从预测失败中提取持久启发式规则（如“检查物体不会移动它”）。
- **选择性预见**（Selective Foresight）：基于置信度（几何平均token概率）过滤不可靠预测，只在足够确信时将预见传递给智能体。

### 关键技术细节
- **情景记忆**（ME）：存储已执行的转换元组 (observation, action, next_observation)，在预测时根据当前候选动作的Jaccard相似度检索Top-kME个最相似历史转换作为上下文证据。
- **语义记忆**（MS）：将预测与真实观察分解为因子化三元组（subject, predicate, object），通过LLM critic从不匹配中生成规则，每条规则附带证据分数，分数正更新；只有正分规则保留在上下文中。可批量更新（kMS个不匹配批次）。
- **选择性预见**：计算生成预测token的几何平均概率q_t，当q_t ≥ τ时传递预测；否则不传递（absention）。
- **算法流程**（Algorithm 1）：
  1. 智能体采样草稿动作a^(0)_t。
  2. 检索情景记忆获得相关转换ME,k。
  3. 世界模型基于当前状态、动作和记忆生成预测õ_{t+1}及置信度q_t。
  4. 选择性过滤：若q_t≥τ则F_t=õ_{t+1}，否则F_t=∅。
  5. 智能体根据(s_t, F_t)采样最终执行动作a_t。
  6. 若最终动作与草稿不同，世界模型对a_t重新查询一次预测。
  7. 执行动作，获取真实观察，更新情景记忆（追加新转换）。
  8. 将预测与真实观察对比，若有差异则通过LLM critic生成规则更新语义记忆。

### 公式
- 预测置信度：ℓ_t = (1/n) Σ log pθ(y_i | ...)，q_t = exp(ℓ_t) ∈ (0,1]。

1. **WorldEvolver在预测准确率上全面领先**：在所有三个骨干模型和两个环境上，WorldEvolver的Exact Match、Token F1、Cosine Similarity均最高。例如Gemma-4-26B-A4B上ALFWorld Exact Match达52.88%（对比Zero-Shot 3.60%，RAWM-ϕ 20.06%）。

## 7. 优点：方法或实验设计上的亮点

- **训练自由（Training-free）**：整个框架不更新任何模型参数，仅通过上下文修订，降低了计算需求和部署复杂性。
- **记忆心理学启发**：清晰区分情景记忆（具体经验）与语义记忆（抽象规则），并赋予不同角色（开发与探索）。
- **选择性预见**：首次将置信度门控应用于世界模型预见，防止不可靠预测误导智能体，类似模型基于强化学习中“何时信任模型”的思想。
- **实验设计全面**：涵盖预测准确率和规划效果两个层面，多种骨干、智能体类型、环境；消融实验完整，超参数分析细致。
- **可解释性**：语义记忆的规则以自然语言形式存储，可审查；记忆更新通过LLM critic完成，过程可控。

## 8. 不足与局限

- **实验覆盖有限**：仅测试了两个文本家庭/科学模拟环境（ALFWorld、ScienceWorld），缺乏对Web导航、代码生成、多模态或物理机器人的评估。方法能否泛化到这些更复杂领域尚未验证。
- **置信度估计依赖token概率**：某些封闭API模型不暴露token logits，此时选择性预见无法直接使用。虽然可用替代方案（如自洽性），但本文未实验。
- **动态阈值选择**：阈值τ需基于校准曲线（图11）预选，不同环境/骨干需单独调整，增加使用成本。
- **记忆扩展性**：随着记忆积累，检索和规则数量可能增加，当前kME=5、kMS=1是手动设定的，在大规模部署下可能需要更高效的记忆管理与检索机制。
- **规则提取依赖LLM critic**：语义记忆依赖LLM生成规则，可能引入噪声或错误规则，证据分数更新机制虽能缓解，但不完善。
- **离线基线设置**：RAWM-ϕ使用了完整Word2World测试集作为检索库（预检索），而WorldEvolver是严格在线积累，比较时对RAWM-ϕ有利，但仍被WorldEvolver超越，进一步凸显后者优势。
