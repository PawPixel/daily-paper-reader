## 摘要
现代大语言模型在后训练阶段依赖强化学习来提升特定能力，然而将多种能力融合到单个模型中仍然困难。现有方法（如离策略微调和混合强化学习）要么效率低下，要么性能损失。本文提出多教师同策略蒸馏（MOPD），一种结合多个领域强化学习教师能力的后训练范式：我们首先对每个领域进行专门的强化学习训练，获得一组领域教师，然后利用学生自身生成的轨迹对这些教师进行蒸馏。这消除了暴露偏差并提供了密集的优化信号。在Qwen3-30B-A3B上，MOPD优于混合强化学习、级联强化学习、离策略微调和参数融合基线，几乎继承了每位教师的全部能力。MOPD还支持领域教师的并行独立开发，消除了多领域后训练中典型的跨领域耦合。MOPD已部署于工业级前沿模型MiMo-V2-Flash的后训练中，展示了其在前沿大语言模型能力融合方面的实用价值。

## Abstract
Modern large language models (LLMs) rely on reinforcement learning during post-training to push specific capabilities, yet integrating multiple capabilities into one model remains hard. Existing methods, such as Off-Policy Finetune and Mix-RL, are either inefficient or lose performance. In this work, we propose Multi-teacher On-Policy Distillation (MOPD), a post-training paradigm for combining the capabilities of multiple domain RL teachers: we first run per-domain specialised RL to obtain a set of domain teachers, then distill these teachers into the student on its own rollouts. This eliminates exposure bias and provides a dense optimization signal. On Qwen3-30B-A3B, MOPD outperforms Mix-RL, Cascade RL, Off-Policy Finetune, and Param-Merge baselines, inheriting nearly all of each teacher's capability. MOPD also enables parallel, independent development of domain teachers, removing the cross-domain coupling typical of multi-domain post-training. MOPD has been deployed in the post-training of MiMo-V2-Flash, an industrial-scale frontier model, demonstrating its practical value for capability integration in frontier-scale LLMs.

### 论文核心问题与整体含义（研究动机和背景）

- **核心问题**：大语言模型（LLM）后训练中，如何将多个领域（如数学推理、指令遵循、软件工程）的专用强化学习（RL）教师的能力高效整合到单个模型中，同时避免能力损失或训练不稳定。
- **背景与动机**：现有方法各有缺陷：
  - **混合RL（Mix-RL）**：混合多领域提示联合训练，但跨领域干扰导致“跷跷板效应”。
  - **级联RL（Cascade RL）**：顺序训练，早期能力衰减，整体训练周期长、稳定性风险高。
  - **离策略微调（Off-Policy Finetune）**：在教师生成的静态数据上微调，遭受暴露偏差（训练与推理分布不匹配）。
  - **参数融合（Param-Merge）**：权重空间平均或任务向量组合，融合后模型不稳定，无法完全继承各教师能力。
- **研究目标**：提出一种在策略空间直接进行多教师能力整合的新范式，兼顾效率、性能和稳定性。

### 论文提出的方法论

- **核心思想**：多教师同策略蒸馏（Multi-teacher On-Policy Distillation, MOPD）。将能力整合从数据空间或权重空间转移到策略空间：学生模型基于自身在线生成（on-policy）的轨迹，接受多个领域教师提供的稠密逐token监督信号，从而继承各教师能力。
- **技术架构（三阶段流水线）**：
  1. **阶段1：通用SFT**：在涵盖所有目标领域的语料上微调基础模型，作为共享初始化。
  2. **阶段2：领域专用RL训练**：针对每个领域独立运行RL（如数学用可验证答案RL，软件工程用沙盒agent RL），得到多个领域专家教师（冻结）。
  3. **阶段3：MOPD蒸馏**：学生从阶段1的SFT检查点初始化，在多领域混合数据集上训练。每一步：
     - 采样一批提示，按领域路由到对应教师；
     - 学生生成轨迹，记录每步token概率；
     - 教师对学生轨迹进行预填充，获得逐token对数概率；
     - 最小化学生与教师在轨迹上的逐token反向KL散度。
- **关键公式与算法细节**：
  - **反向KL损失**：
    \[ \mathcal{L}_{\text{rev-KL}} = \mathbb{E}_{x,y\sim\pi_\theta} \left[ \frac{1}{|y|} \sum_t \sum_v \pi_\theta(v) \log\frac{\pi_\theta(v)}{\pi_{\phi_d}(v)} \right] \]
    其中 \(\pi_{\phi_d}\) 是提示 \(x\) 对应的领域教师。
  - **策略梯度实现**：将上述损失转化为策略梯度形式，其中优势函数为 \(\hat{A}_t = \log \pi_{\phi_d}(y_t) - \log \pi_\theta(y_t)\)，并应用双边截断。
  - **Top-\(k\) 蒸馏实现**：只对教师的前 \(k\) 个高概率token计算损失，额外添加校正项 \(\pi_{\phi_d}(v) - \pi_\theta(v)\) 保证最优解正确，降低通信与计算开销。
- **基础设施**：将领域教师部署为独立预填充服务，与采样异步重叠，几乎不增加墙钟时间。
- **核心优势**：
  - 消除暴露偏差（学生训练分布即推理分布）；
  - 稠密逐token监督（比轨迹级奖励方差更低、样本效率更高）；
  - 策略空间融合稳定（路由到特定教师而非权重平均）；
  - 模块化并行开发（各教师独立训练，互不影响）；
  - 同源教师保证初始KL小，优化稳定。

### 论文的主要结论与发现

1. **MOPD优于所有基线**：在Qwen3-30B-A3B上归一化分数达0.937，领先第二名Mix-RL（0.882）5.5个点。
2. **均匀继承教师能力**：三个领域的归一化分数范围仅0.044（0.91-0.95），其他基线存在显著波动（如Cascade RL范围0.41）。
3. **样本效率高**：MOPD在IF和SWE上约25k-30k样本即达到教师水平，Mix-RL需要150k-180k。
4. **同源教师是关键**：替换为更强但分布不同的外部教师（Qwen3-235B-A22B）导致训练不稳定或崩溃，表明same-origin教师提供低初始KL，保障优化稳定性。
5. **多轮迭代可进一步提升**：第二轮MOPD（基于第一轮学生重新训练教师）归一化分数从0.937升至0.986。
6. **可扩展至工业级模型**：在MiMo-V2-Flash（309B）上匹配或超越各教师，大部分benchmark有增益。

### 优点

- **方法创新性**：将多能力整合从离线数据/权重空间转向在线策略蒸馏，兼顾稠密监督与暴露偏差消除。
- **工程实用性**：支持教师并行开发、风险隔离、配方解耦，降低多领域后训练的组织复杂度。
- **实验严谨性**：使用归一化分数避免不同领域量纲差异；对比了四类代表性基线；消融实验验证了两个关键设计（损失形式、教师来源）。
- **可扩展性验证**：在中等规模（30B MoE）和工业级（309B）模型上均取得良好结果。

### 不足与局限

- **实验覆盖不足**：
  - 仅在两个模型上验证（Qwen3-30B-A3B和MiMo-V2-Flash），后者未公开细节，通用性需更多架构验证。
  - 未消融top-\(k\)中不同\(k\)值的影响；未探讨混合RL不同领域比例对基线的影响。

- **潜在偏差风险**：
  - 蒸馏过程可能放大教师中的偏差或错误；缺乏对安全性、偏见等维度的评估。

- **应用限制**：需要维护多个教师推理服务，增加了基础设施复杂度；对于资源紧张的团队，并行训练多个领域教师可能仍有开销。
