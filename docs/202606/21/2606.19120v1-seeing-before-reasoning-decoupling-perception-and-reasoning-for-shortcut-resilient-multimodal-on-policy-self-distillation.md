## 摘要
在策略自蒸馏（OPSD）在自身 rollout 上训练模型，并使用冻结副本提供以参考目标为条件的密集 token 级目标。这适用于大语言模型（LLM）推理，但直接扩展到多模态大语言模型（MLLMs）可能会产生捷径：特权目标可能主要基于文本参考目标而非图像来引导 token。我们提出 ViGOS，一种面向 MLLM 后训练的视觉引导在策略自蒸馏框架。学生模型先撰写视觉描述，再推理出最终答案。对于有效的 rollout，仅图像感知教师监督描述，而特权推理教师在同一学生前缀上监督推理和最终答案。仅对无效 rollout 使用参考教师以恢复输出格式。在通用视觉-语言、专家推理、视觉数学、空间定位和视觉-语言先验基准上，ViGOS 保持了 OPSD 的主要优势，并在易产生捷径的设置中改善了基于图像的行为。

## Abstract
On-policy self-distillation (OPSD) trains a model on its own rollouts and uses a frozen copy to provide dense token-level targets conditioned on a reference target. This works well for LLM reasoning, but a direct extension to multimodal large language models (MLLMs) can create a shortcut: the privileged target may guide tokens mainly based on the text reference target rather than the image. We propose ViGOS, a visually grounded OPSD framework for MLLM post-training. The student first writes a visual description and then reasons toward the final answer. For valid rollouts, an image-only perception teacher supervises the description, while a privileged reasoning teacher supervises the reasoning and final answer on the same student prefix. A reference teacher is used only for invalid rollouts to recover the output format. Across general vision-language, expert reasoning, visual math, spatial grounding, and visual-language-prior benchmarks, ViGOS keeps the main benefits of OPSD and improves image-grounded behavior in shortcut-prone settings.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **背景**：On-policy self-distillation（OPSD）在大语言模型（LLM）推理上表现优异，它通过在自身 rollout 上训练模型，并使用冻结的教师模型提供密集的 token 级监督信号（以参考答案为条件）。
- **核心问题**：直接将 OPSD 扩展到多模态大语言模型（MLLMs）会引发一个“捷径”问题：特权教师模型在监督时能看到参考答案，而文本信号比图像更容易被模型利用，导致学生模型可能主要依赖文本参考答案而非图像内容来生成正确的推理过程和答案，造成弱视觉接地（weak visual grounding）。
- **整体意义**：提出 ViGOS（Visual Grounding On-Policy Self-Distillation），通过将感知（perception）与推理（reasoning）解耦，让模型先写视觉描述再推理，从而抑制捷径问题，提升 MLLM 的图像接地能力。

## 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：将学生的生成序列结构化分为三个部分：视觉描述（description）、推理过程（reasoning）、最终答案（answer）。使用三种不同的冻结教师模型对不同部分进行监督，隔离参考答案对早期视觉描述的直接影响。
- **关键技术细节**：
  - **学生 rollout**：学生模型接收图像 I 和问题 x（不含参考答案），生成带标签的结构化输出：`<description> d </description> <think> r </think> \boxed{a}`。
  - **三种教师角色**（均由同一初始化模型冻结得到）：
    1. **图像仅感知教师（image-only perception teacher, q_img）**：仅输入图像，不输入问题或参考答案，监督描述 token 段（T_d）。损失为 𝓛_perc = 𝔼[ (1 - m_inv) Σ D_KL(q_img,t ∥ p_θ,t) ]。
    2. **特权推理教师（privileged reasoning teacher, q_rea）**：输入图像、问题、参考答案，监督推理和答案 token 段（T_r ∪ T_a）。损失为 𝓛_rea = 𝔼[ (1 - m_inv) Σ D_KL(q_rea,t ∥ p_θ,t) ]。
    3. **参考教师（reference teacher, q_ref）**：输入图像、问题、参考答案，但仅在 rollout 无效（格式错误等）时使用，监督全部 token，采用反向 KL 散度。损失为 𝓛_ref = 𝔼[ m_inv Σ D_KL(p_θ,t ∥ q_ref,t) ]。
  - **总损失**：𝓛_ViGOS = λ_perc 𝓛_perc + λ_rea 𝓛_rea + λ_ref 𝓛_ref。有效 rollout 时只前两项激活，无效时仅最后一项激活。
- **算法流程**（文字说明）：
  1. 从数据集采样 (I, x, a*)。
  2. 学生根据 (I, x) 采样结构化响应 y。
  3. 解析器提取描述 token 集 T_d、推理 token 集 T_r、答案 token 集 T_a，并判断是否有效。
  4. 对有效 rollout：使用图像仅感知教师计算 T_d 上 KL 散度；使用特权推理教师计算 T_r∪T_a 上 KL 散度。
  5. 对无效 rollout：使用参考教师计算全部 token 上的反向 KL 散度。
  6. 反向传播更新学生（只更新可训练学生，教师冻结）。

## 6. 论文的主要结论与发现

- ViGOS 在保持 OPSD 标准基准增益的同时，显著改善了先验敏感场景下的图像接地行为。
- 在 3B 和 7B 上，ViGOS 的平均 Pass@5 均高于 Baseline，在多数任务上与 OPSD 持平或更好。
- 在 ViLP 上，ViGOS 获得最高的 Score（图像支持正确答案），同时保持高 Prior（先验正确时仍能正确回答），表明其不单纯抑制先验，而是更好地平衡图像与文本。

- 训练动态显示：ViGOS 的 ViLP Score 持续上升，而 OPSD 先升后降，说明解耦设计避免了捷径影响。
